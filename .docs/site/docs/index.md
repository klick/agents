# Agents Plugin Documentation

Public technical documentation for the `klick/agents` Craft CMS plugin.

This docs site is optimized for:

- integration engineers
- operations teams
- API clients and LLM/crawler consumption

## What this plugin provides

- read-only API surface for health, readiness, products, orders, entries, sections, and changes
- public discovery surfaces: `/llms.txt` and `/commerce.txt`
- scoped token auth and operational security controls
- optional webhook delivery for resource change events
- Craft CLI commands for discovery and diagnostics
- control panel cockpit for runtime operations and posture visibility
